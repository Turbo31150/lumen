> Application web React 19 conçue pour fonctionner en tandem avec le cluster IA local **JARVIS**. Elle centralise en un seul outil tout ce qu'un professionnel ou un apprenant a besoin pour travailler dans n'importe quelle langue, en temps réel.

---

## ✨ Fonctionnalités

| Feature | Description |
|---|---|
| 🎤 Transcription live | Capture micro → WhisperFlow (Whisper large-v3) |
| 🌐 Traduction simultanée | Multi-langues en temps réel via IA |
| ✏️ Correction grammaticale | Correction IA contextuelle |
| 🔊 Synthèse vocale | TTS Piper haute-fidélité |
| 📱 Interface moderne | React 19 responsive + WebSocket |

---

## 🛠️ Stack Technique

```
Frontend : React 19 + Vite + TailwindCSS
Backend  : Node.js + Express + WebSocket
IA       : WhisperFlow (transcription) + JARVIS (traduction/TTS)
Infra    : Docker + docker-compose
```

---

## 🚀 Démarrage rapide

```bash
git clone https://github.com/Turbo31150/lumen
cd lumen
docker-compose up -d
# Ouvrir http://localhost:3000
```
